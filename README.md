## Excursion Landing Page
This is a simple landing page for the fictional mobile app Excursion, built using only HTML and CSS.

## How It Was Developed
The project was created using HTML for structure and CSS for styling.
A video, images, and icons are included to match the design mock.
File structure follows best practices using separate folders for CSS, images, and videos.
A fixed footer is added using CSS.
Git was used for version control, and the project was pushed to GitHub.
Deployed using GitHub Pages.
